# Playwright Automation Framework

This repository contains Playwright automation test scripts using JavaScript.

## Technologies Used
- Playwright
- JavaScript
- Node.js

## Features Covered
- Page Object Model (POM)
- File Upload
- Shadow DOM
- Web Tables
- Pagination
- Alerts Handling
- Multiple Windows
- Dropdown Handling
- Dynamic Web Elements

## Project Structure

```text
Automation_Project1
│
├── Pages
├── tests
├── package.json
├── playwright.config.js
└── README.md
```

## Run Tests

```bash
npx playwright test
```

## Open HTML Report

```bash
npx playwright show-report
```
